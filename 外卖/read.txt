# FlavorDash 外卖系统

## 项目介绍
FlavorDash是一个现代化的外卖订餐系统，提供完整的点餐、购物车、订单管理和支付功能。系统采用前后端分离架构，前端使用HTML、CSS和JavaScript，后端使用Node.js和MySQL。

## 功能特性

### 核心功能
- 🌐 独立的登录页面（微信和QQ一键登录）
- 📋 菜品浏览和分类筛选
- 🛒 购物车管理（可拖拽）
- 📍 订单配送地址管理
- 💳 微信支付模拟
- 📊 订单历史记录
- 👤 个人中心
- 🔧 后台管理（菜品上架）
- 💬 智能客服（DeepSeek API）

### 技术亮点
- 响应式设计，适配不同设备
- 现代化的UI界面，流畅的动画效果
- 完整的用户认证系统
- 安全的会话管理
- 模块化的代码结构
- 实时数据更新

## 技术栈

### 前端
- HTML5
- CSS3
- JavaScript (ES6+)
- Remix Icon (图标库)

### 后端
- Node.js
- Express.js
- MySQL
- DeepSeek API (智能客服)

```
外卖/
```

## 安装步骤

### 1. 环境要求
- Node.js 14.0+
- MySQL 5.7+

### 2. 克隆项目
```bash
git clone <repository-url>
cd 外卖
```

### 3. 安装依赖
```bash
npm install
```

### 4. 配置数据库
- 创建MySQL数据库：`food_delivery_db`
- 修改`server.js`中的数据库连接信息（如果需要）

### 5. 初始化数据库
```bash
node init_db.js
```

### 6. 启动服务器
```bash
node server.js
```

## 使用说明

### 访问系统
打开浏览器，访问：`http://localhost:8080`

### 登录流程
1. 进入登录页面
2. 选择微信或QQ一键登录
3. 登录成功后进入主页面

### 点餐流程
1. 浏览菜品，可按分类筛选
2. 点击菜品添加到购物车
3. 在购物车中调整数量
4. 点击"立即支付"
5. 填写配送地址
6. 扫描二维码支付
7. 确认支付成功

### 后台管理
- 点击侧边栏"后台"按钮
- 填写菜品信息并上架

## 注意事项

1. **数据库配置**：确保MySQL服务正在运行，且数据库连接信息正确
2. **端口占用**：默认使用8080端口，若被占用请修改`server.js`中的端口设置
3. **网络连接**：智能客服功能需要网络连接
4. **安全提示**：当前为开发环境，生产环境需要添加更多安全措施
